## 1. Academic Style

**Expected:** High academic score, complex vocabulary, long sentences, noun-heavy

```
The epistemological foundations of contemporary cognitive science necessitate a thorough examination of the relationship between neural substrates and phenomenological experience. Furthermore, the integration of computational models with empirical neuroscientific data has yielded unprecedented insights into the mechanisms underlying human cognition. These interdisciplinary approaches have fundamentally transformed our understanding of consciousness and its neural correlates, establishing new paradigms for investigating the mind-brain relationship.
```

## 16. Philosophical Style

**Expected:** High academic score, abstract nouns, rhetorical questions, hedging

```
What does it mean to act freely in a world governed by causal necessity? If every decision emerges from prior states of the brain, which themselves follow from earlier physical events, then the very concept of agency becomes philosophically suspect. Yet we cannot escape the phenomenology of choice — the felt sense that alternatives genuinely exist before we select among them. Perhaps freedom is not the absence of causation but a particular kind of self-caused determination, one in which the reasoning self is the proximate cause of its own actions.
```

```

```

### 27. Indeterminate Style (Neutral text)

**Expected:** Low scores across all categories, possibly "Indeterminate"

```
Items are on the table. The door is open. Water is in the glass. Books are on the shelf. The light is on. Papers are in the folder. Keys are in the drawer. Food is in the fridge.
```

---

### 28. Code-Heavy Text

**Expected:** Technical score dominant, possibly flagged as non-prose or indeterminate

```
function calculateSum(arr) { return arr.reduce((acc, val) => acc + val, 0); } const result = calculateSum([1, 2, 3, 4, 5]); console.log(result); if (result > 10) { return true; } else { return false; }
```

---

### 29. Numbers and Data Heavy (Low Verbal Signal)

**Expected:** Low confidence or indeterminate — minimal stylistic markers

```
Total: 4,821. Variance: 0.034. Mean: 72.6. Std Dev: 12.4. Median: 71.0. Min: 48. Max: 99. N: 300. p-value: 0.002. CI: [68.2, 77.0]. R²: 0.81.
```

### 30. Multilingual / Code-Switched Text

**Expected:** Low confidence, possibly indeterminate — analyzer stress test

```
So basically I was thinking, ¿sabes? It's complicated to explain pero I'll try. The situation is muy difícil right now. Honestly, ne znam što da radim. People just don't understand, you know what I mean? C'est la vie, I guess.
```

**Expected:** Persuasive or creative score, possibly with low confidence — interrogative-heavy edge case

```
Have you ever wondered why things turn out the way they do? What if everything you believed was slightly wrong? Does meaning come from the world or from the stories we tell about it? Who decides what matters? Can we ever really know another person completely? What would you do differently if you weren't afraid?
```
